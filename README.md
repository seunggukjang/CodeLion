## 서버란?
* 서버(호스트)는 클라이언트(게스트)에게 적절한 서비스를 제공하는 단말기나 프로그램입니다. 인터넷에선 TCP같은 프로토콜로 클라이언트가 서버에게 연결 요청을 보낸 뒤 응답과 반응을 통해 서비스를 제공하며, 요청-반응 같은 관계를 서버-클라이언트 관계라고도 합니다.
